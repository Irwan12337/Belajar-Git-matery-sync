Belajar git

Latihan GIt

today ge gonna learn basic aboyt git
Command untuk add : git add index.html
command untuk add keseluruhan : git add .
command untuk menambahkan author identity : git config --global user.email "you@example.com, untuk nama tinggal ganti user.name
command untuk commit : git commit -m "Judul bebas"
command untuk melihat apa saja perubahan dalam script : git diff
command untuk melihat log : git log
command untuk menghapus/revisi file : git rm --cached index.html
command untuk connect ke git hub : git remote add origin "Url yang dicopy dari github"
command untuk melihat fungsi github : git remote -v
command untuk upload source code ke github : git push origin master

Git branch

untuk melihat file branch : git branch, dan untuk menambahkan branch baru : git branch "diikuti nama file
untuk berpindah folder branch : git checkout "diikuti nama file", untuk kembali ke master : git checkout master
untuk mengahpus file branch : git branch -d "nama file"
merge branch : git merge "nama file branch" sebelum merge pindah dahulu ke checkout master
berpindah dan create brach dengan cepat : git checkout -b feat/"nama file branch"
belajar git

Git tag

untuk menambahkan tag ketik git tag -a "namafile", lalu tambahkan catatan kemudian klik ctrl+X then Y
untuk cek ketik : git tag
untuk mengupload : git push origin master --tags
untuk menghapus : git tag -d "nama file"

Git Lanjutan

cara mengembalikan source code ke working directory (melepas staging) : git restore -S "nama file"
mengembalikan source code yang telah di commit ke working directory kembali : git reset --hard "kemudian masukkan nomor hash, nomor yang didapat dari git log"
mengembalikan source code ke commit nomor 3 dari 5 commit yang telah dibuat tanpa menghapus commit 4 & 5 ketik : git revert "nomor hash yang di show di log"
clone source code dari github : git clone "copy code dari repository git"
