Latihan GIt

today ge gonna learn basic aboyt git
Command untuk add : git add index.html
command untuk add keseluruhan : git add .
command untuk menambahkan author identity : git config --global user.email "you@example.com, untuk nama tinggal ganti user.name
command untuk commit : git commit -m "Judul bebas"
command untuk melihat apa saja perubahan dalam script : git diff
command untuk melihat log : git log
command untuk menghapus/revisi file : git rm --cached index.html
command untuk connect ke git hub : git remote add origin "Url yang dicopy dari github"
command untuk melihat fungsi github : git remote -v
command untuk upload source code ke github : git push origin master

Git branch

untuk melihat file branch : git branch, dan untuk menambahkan branch baru : git branch "diikuti nama file
untuk berpindah folder branch : git checkout "diikuti nama file", untuk kembali ke master : git checkout master
untuk mengahpus file branch : git branch -d "nama file"
